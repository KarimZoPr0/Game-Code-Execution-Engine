# Game Code Execution Engine

A comprehensive game development platform featuring a web-based code editor and execution engine powered by WebAssembly.

## Live Demo
[Game Code Execution Engine](https://karimzopr0.github.io/Game-Code-Execution-Engine/)

## Getting Started

### Frontend
```bash
cd frontend
docker-compose up
```

### Backend
```bash
cd backend
docker-compose up
```

## Features

- Mobile-responsive interface
- Code execution system
- Live coding
- Multiplayer
- Timeline

## Tech Stack

- **Frontend**: React, TypeScript, Vite, Monaco Editor, FlexLayout
- **Backend**: Node.js, Express, Emscripten, WebSocket
